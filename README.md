# Heros & AG-Grid Study

* 專案文件下載後請移至專案目錄並安裝套件，請輸入以下指令：

  ```
  npm install
  ```

* 開啟angular server指令：
  
  ```
  ng serve --open
  ```

## Study STEP

* STEP 1. 完成[英雄之旅應用程式和教程](https://angular.tw/tutorial/tour-of-heroes)


* STEP 2. 完成[AG Grid 應用](https://www.ag-grid.com/)

* STEP 3. 完成新增英雄功能、顯示清單功能並使用AG Grid套件
